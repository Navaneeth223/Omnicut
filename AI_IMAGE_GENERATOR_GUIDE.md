## How to Use (4-Step Workflow)

### Step 1: Write Your Prompt (1 minute)

Describe the image you want to create in detail:

**Good Prompt Example:**
```
A majestic dragon flying over a medieval castle at sunset, 
cinematic lighting, highly detailed, fantasy art style, 
vibrant colors, dramatic clouds, 8k quality
```

**Tips for Better Prompts:**
- Be specific and descriptive
- Include art style (e.g., "digital art", "oil painting", "photorealistic")
- Mention lighting and mood
- Add quality keywords ("highly detailed", "8k", "professional")
- Describe colors and atmosphere

**Negative Prompt (Optional):**
Describe what you DON'T want:
```
blurry, low quality, distorted, ugly, bad anatomy, 
watermark, text, signature
```

### Step 2: Choose Settings (30 seconds)

#### Select AI Backend

We support **6 free AI backends**:

1. **Pollinations AI** ⚡ RECOMMENDED
   - ✅ Completely free, no API key required
   - ⚡ Fast generation (10-30 seconds)
   - ✨ Great quality
   - 🎯 Best for: Quick iterations, testing prompts

2. **HuggingFace (SDXL)** 🏆 BEST QUALITY
   - ✅ Free tier available
   - 🔑 Requires free API key
   - ⚡ Medium speed (30-60 seconds)
   - ✨ Excellent quality (Stable Diffusion XL)
   - 🎯 Best for: High-quality final images

3. **Replicate** 🎨
   - ✅ Free credits available
   - 🔑 Requires API key
   - ⚡ Medium speed
   - ✨ Excellent quality, multiple models
   - 🎯 Best for: Advanced users, specific models

4. **Craiyon (DALL-E Mini)** 🎭
   - ✅ Completely free, no signup
   - ⚡ Slower (60-120 seconds)
   - ✨ Good quality
   - 🎯 Best for: Fun, experimental images

5. **DeepAI** 💫
   - ✅ Free tier with API key
   - 🔑 Requires free API key
   - ⚡ Fast generation
   - ✨ Great quality
   - 🎯 Best for: Quick professional images

6. **Stability AI** 🚀
   - ✅ Free trial credits
   - 🔑 Requires API key
   - ⚡ Medium speed
   - ✨ Excellent quality (official Stable Diffusion)
   - 🎯 Best for: Professional projects

#### Select Aspect Ratio

Choose the format that matches your needs:

- **Square (1:1)** - 1024×1024
  - Perfect for: Instagram posts, profile pictures, thumbnails
  
- **Portrait (9:16)** - 768×1344
  - Perfect for: TikTok, Instagram Reels, phone wallpapers
  
- **Landscape (16:9)** - 1344×768
  - Perfect for: YouTube thumbnails, desktop wallpapers, presentations
  
- **Wide (21:9)** - 1536×640
  - Perfect for: Banners, headers, cinematic shots

### Step 3: Generate! (10-120 seconds)

Click "✨ Generate Image" and wait while AI creates your masterpiece!

**Generation Times:**
- Pollinations: 10-30 seconds ⚡
- HuggingFace: 30-60 seconds
- DeepAI: 10-30 seconds ⚡
- Replicate: 30-60 seconds
- Craiyon: 60-120 seconds 🐌
- Stability AI: 30-60 seconds

### Step 4: Save & Use (10 seconds)

Once generated:
- **Preview** your image
- **Save to Media Pool** to use in your projects
- **Download** to your computer
- **Generate another** with different settings

## Prompt Engineering Tips

### Basic Structure
```
[Subject] + [Action] + [Setting] + [Style] + [Quality]
```

### Examples

#### Fantasy Art
```
Prompt: A mystical forest with glowing mushrooms and fireflies, 
magical atmosphere, ethereal lighting, fantasy art style, 
highly detailed, vibrant colors, 8k quality

Negative: blurry, low quality, realistic, photograph
```

#### Photorealistic Portrait
```
Prompt: Professional portrait of a confident businesswoman, 
modern office background, natural lighting, photorealistic, 
sharp focus, 50mm lens, high resolution

Negative: cartoon, anime, painting, blurry, distorted face
```

#### Sci-Fi Scene
```
Prompt: Futuristic cyberpunk city at night, neon lights, 
flying cars, rain-soaked streets, cinematic composition, 
dramatic lighting, highly detailed, 8k quality

Negative: daytime, bright, cheerful, low quality, blurry
```

Prompt: Abstract geometric patterns in vibrant colors, 
modern art style, flowing shapes, gradient colors, 
minimalist design, high contrast, 4k quality

## Troubleshooting

### "API Key Required"
**Solution:** 
1. Click "Enter API Key" button
2. Get free API key from backend's website
3. Paste into dialog
4. Key is saved for future use

### "Generation Failed"
**Possible Causes:**
- API key invalid or expired
- Backend service temporarily down
- Prompt contains restricted content
- Network connection issue

**Solutions:**
- Try different backend
- Check API key is correct
- Simplify prompt
- Check internet connection

### "Image Quality is Low"
**Solutions:**
- Use HuggingFace SDXL for best quality
- Add quality keywords to prompt
- Increase image size (aspect ratio)
- Be more specific in description

### "Generation is Slow"
**Solutions:**
- Use Pollinations AI for fastest results
- Try DeepAI for quick generation
- Avoid Craiyon if speed is priority
- Be patient with high-quality backends

### "Image Doesn't Match Prompt"
**Solutions:**
- Be more specific and detailed
- Add style keywords
- Use negative prompts
- Try different backend
- Iterate and refine prompt

## FAQ

**Q: Is it really free?**
A: Yes! All backends offer free tiers. Some require free API keys but no payment.

**Q: How many images can I generate?**
A: Depends on backend:
- Pollinations: Unlimited
- HuggingFace: 1,000/month
- Replicate: ~100-200 with free credits
- Craiyon: Unlimited
- DeepAI: 100/month
- Stability AI: ~100-200 with trial credits

**Q: Can I use generated images commercially?**
A: Check each backend's terms of service. Most allow commercial use with attribution.

**Q: Which backend is best?**
A: 
- For speed: Pollinations or DeepAI
- For quality: HuggingFace SDXL or Stability AI
- For no signup: Pollinations or Craiyon

**Q: Can I generate specific people or characters?**
A: Avoid copyrighted characters and real people's names. Use generic descriptions instead.

**Q: How do I improve image quality?**
A: Use detailed prompts, quality keywords, and high-quality backends like HuggingFace SDXL.

**Q: Can I edit generated images?**
A: Yes! Save to Media Pool and use in the editor with effects and adjustments.

**Q: What if generation fails?**
A: Try a different backend, check your API key, or simplify your prompt.
