## 🎯 Complete Workflow

### **1. Import Media**
```
File → Import Media (⌘I)
OR Click 📁 in Media Pool
OR Drag files into Media Pool

Supported formats:
- Video: MP4, MOV, WebM, AVI
- Audio: MP3, WAV, AAC, OGG
- Image: JPG, PNG, GIF, WebP
```

### **2. Add to Timeline**
```
Drag from Media Pool → Drop on track
OR Double-click file in Media Pool

Features:
- Automatic track matching
- Visual feedback (blue highlight)
- Position at drop location
- Thumbnail on clip
```

### **3. Edit Clips**
```
Trim: Drag clip edges
Split: Press C at playhead
Move: Drag clip
Copy: Cmd+C
Paste: Cmd+V
Delete: Delete key

Advanced:
- Multi-selection (Cmd+Click)
- Magnetic snapping
- Ripple editing
- Collision detection
```

### **4. Apply Effects**
```
1. Select clip on timeline
2. Open Effects panel (right side)
3. Browse or search effects
4. Click effect to apply
5. Adjust parameters with sliders
6. See changes in real-time in viewer ✨
7. Toggle on/off with eye icon
8. Remove with X button
```

### **5. Add Transitions**
```
1. Select 2 adjacent clips
2. Open Transitions panel (right side)
3. Set duration (0.1-5 seconds)
4. Click transition to apply
5. View in applied transitions list
6. Remove if needed
```

### **6. Preview**
```
Click timeline to seek
Press Space to play
Use ←/→ to step frames
See video with effects in viewer ✨
Adjust playback speed
Use viewer controls
```

### **7. Export Video** ✨
```
File → Export Video (⌘E)
OR Click Export button

1. Choose format (MP4, WebM, MOV)
2. Select codec (H.264, H.265, VP9)
3. Set resolution (1920x1080, 3840x2160, etc.)
4. Choose quality (Low, Medium, High, Ultra)
5. Configure audio settings
6. Click Export
7. Wait for FFmpeg to render
8. Video downloads automatically

Features:
- Real FFmpeg rendering
- Effects baked into video
- Transitions rendered
- Progress tracking
- Accurate file size
- Multiple formats
```
