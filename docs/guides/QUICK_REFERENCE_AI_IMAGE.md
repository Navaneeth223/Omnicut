# 🎨 AI Image Generator - Quick Reference

## 🚀 Quick Start (30 seconds)

1. Click **"🎨 AI Image"** tab
2. Enter prompt: `"A beautiful sunset over mountains, cinematic, 8k"`
3. Choose **Pollinations AI** (no API key needed)
4. Click **"✨ Generate Image"**
5. Wait 10-30 seconds
6. Click **"💾 Save to Media Pool"**

**Done!** Your AI image is ready to use.

## 📐 Aspect Ratios

- **Square (1:1)** → Instagram posts, thumbnails
- **Portrait (9:16)** → TikTok, Reels, Stories
- **Landscape (16:9)** → YouTube, presentations
- **Wide (21:9)** → Banners, headers

---

## ✍️ Prompt Formula

```
[Subject] + [Action] + [Setting] + [Style] + [Quality]
```

**Example:**
```
A dragon flying over a castle at sunset,
cinematic lighting, fantasy art, highly detailed, 8k
```

---

## 🎯 Power Keywords

**Quality**: highly detailed, 8k quality, professional, masterpiece

**Lighting**: cinematic lighting, dramatic lighting, golden hour

**Style**: digital art, oil painting, photorealistic, fantasy art

**Mood**: ethereal, dramatic, peaceful, vibrant, moody

---

## ⚡ Speed Tips

1. **Use Pollinations** for fastest results (10-30s)
2. **Be specific** in prompts (faster processing)
3. **Avoid complex** negative prompts
4. **Choose smaller** aspect ratios for speed

---

## 🔑 Getting API Keys

### HuggingFace (Best Quality)
1. [huggingface.co](https://huggingface.co) → Sign up
2. Settings → Access Tokens → Create
3. Copy → Paste in OmniCut
4. **Free**: 1,000 images/month

### DeepAI (Fast)
1. [deepai.org](https://deepai.org) → Sign up
2. Dashboard → API Keys → Copy
3. Paste in OmniCut
4. **Free**: 100 images/month

## 🐛 Troubleshooting

**"API Key Required"**
→ Click "Enter API Key" → Get free key from backend website

**"Generation Failed"**
→ Try different backend or simplify prompt

**"Low Quality"**
→ Use HuggingFace SDXL + add quality keywords

**"Slow Generation"**
→ Use Pollinations or DeepAI for speed
