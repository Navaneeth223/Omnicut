## 🎯 Complete Workflow

### **1. Import Media**
```
File → Import Media (⌘I)
OR
Click 📁 in Media Pool
OR
Drag files into Media Pool
```

### **2. Add to Timeline**
```
Drag from Media Pool → Drop on track
OR
Double-click file in Media Pool
```

### **3. Edit**
```
- Trim: Drag clip edges
- Split: Press C at playhead
- Move: Drag clip
- Copy: Cmd+C
- Paste: Cmd+V
- Delete: Delete key
```

### **4. Apply Effects**
```
1. Select clip
2. Open Effects panel
3. Browse/search effects
4. Click to apply
5. Adjust parameters
```

### **5. Add Transitions**
```
1. Select 2 adjacent clips
2. Open Transitions panel
3. Set duration
4. Click transition
```

### **6. Preview**
```
- Click timeline to seek
- Press Space to play
- See video in viewer
- Use ←/→ to step frames
```

### **7. Export**
```
File → Export Video (⌘E)
OR
Click Export button
```
