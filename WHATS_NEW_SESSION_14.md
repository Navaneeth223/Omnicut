## 🚀 How to Use

### Quick Start (2 minutes)

1. **Open AI Image Workspace**
   - Click "🎨 AI Image" tab in header

2. **Enter Prompt**
   ```
   A majestic dragon flying over a medieval castle at sunset,
   cinematic lighting, highly detailed, fantasy art style
   ```

3. **Choose Settings**
   - Backend: Pollinations AI (no API key needed!)
   - Aspect Ratio: Landscape (16:9)

4. **Generate**
   - Click "✨ Generate Image"
   - Wait 10-30 seconds

5. **Save & Use**
   - Click "💾 Save to Media Pool"
   - Use in AI Shorts Studio
   - Export video!

**Total Time**: ~2 minutes per image!

## 💡 Pro Tips

### Writing Better Prompts

**Good Prompt Structure:**
```
[Subject] + [Action] + [Setting] + [Style] + [Quality]
```

**Example:**
```
A mystical forest with glowing mushrooms and fireflies,
magical atmosphere, ethereal lighting, fantasy art style,
highly detailed, vibrant colors, 8k quality
```

**Power Keywords:**
- Quality: "highly detailed", "8k quality", "professional"
- Lighting: "cinematic lighting", "dramatic lighting", "golden hour"
- Style: "digital art", "oil painting", "photorealistic"
- Mood: "ethereal", "dramatic", "peaceful", "vibrant"

### Negative Prompts

Tell AI what you DON'T want:
```
blurry, low quality, distorted, ugly, bad anatomy,
watermark, text, signature
```
